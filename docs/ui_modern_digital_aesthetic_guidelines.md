# UI Modern Digital Aesthetic Guidelines

The target product feel is a calm memory operating system:

- card-first layout,
- restrained color,
- clear status badges with text labels,
- generous spacing,
- progressive disclosure for technical details,
- no benchmark-score panels in the primary UI,
- no raw JSON walls on default pages.

State color intent:

- green: active or verified,
- purple: canonical or compacted intelligence,
- amber: deprecated or needs review,
- red: contradicted or destructive action,
- gray: archived, superseded, or metadata.

All state indicators should include text; color alone is not sufficient.
