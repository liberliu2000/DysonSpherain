# Optional Image2 Prompts

The LaTeX bundle already contains completed TikZ/PGFPlots figures. If raster
images are required for a visual submission package, generate replacements from
the prompts below and save them under `paper/latex/figures/raster/`.

## System Overview

Create a clean academic systems diagram for a paper titled "DysonSpherain".
Show a left-to-right pipeline with boxes for CLI, Memory OS Interface, Research
Evaluation Harness, Experiment Registry, Runtime Context Compiler, Retrieval
Pipeline and Diagnostics, and Formal Protocol Validator. Use a restrained
white-background NeurIPS-style figure, black text, thin gray arrows, and no
decorative gradients.

## Main Quality Results

Create a publication-ready grouped bar chart on a white background. X-axis:
LongMemEval, LoCoMo, KnowMe, CloneMem. Two bars per benchmark: Final R@10 and
Final NDCG@10. Values: LongMemEval 0.9778 and 0.9259; LoCoMo 0.9070 and 0.7533;
KnowMe 0.5972 and 0.5051; CloneMem 0.0954 and 0.0752. Use high-contrast but
muted colors, gridlines, and readable labels.

## Efficiency Sweeps

Create a publication-ready bar chart on a white background. X-axis categories:
candidate top-50, candidate top-100, candidate top-200, candidate top-500,
rerank top-20, rerank top-50, rerank top-100. Y-axis elapsed seconds. Values:
85.964, 86.081, 86.675, 88.789, 91.702, 93.427, 93.807. Use restrained
academic styling with value labels above bars.
